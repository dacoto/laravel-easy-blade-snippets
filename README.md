| blade-forelse           | @forelse...@empty...@endforelse                          |
| blade-while             | @while...@endwhile                                       |
| blade-each              | @each                                                    |

| blade-stack             | @stack                                                   |

| blade-comment           | {{-- comment --}}                                        |
| blade-echo              | {{ $data }}                                              |
